# Sticks Fighters

Sticks Fighters é um jogo de luta 2D desenvolvido em Java como projeto de estudo e prática de programação orientada a objetos, testes de software e desenvolvimento de jogos.

O projeto começou como uma implementação simples utilizando personagens em formato de stickman e evoluiu para uma versão com personagens próprios, sprites animados e mecânicas básicas de combate.

## Funcionalidades atuais

* Seleção de personagens
* Movimentação lateral
* Sistema de pulo
* Sistema de agachamento
* Ataques corpo a corpo
* Sistema de vida
* Tela de Game Over
* Menu de pausa
* Sprites animados
* Estrutura modularizada em pacotes
* Testes automatizados para partes do sistema

## Personagens

### Chicão

Personagem jogável com sprites próprios para:

* Idle
* Caminhada
* Pulo
* Agachamento
* Soco
* Chute
* Vitória
* Receber dano

### Rata Camponesa

## Como executar

1. Clone o repositório:

git clone https://github.com/AninhaLeao-B/Sticks-Fighters.git

2. Abra o projeto em sua IDE de preferência.

3. Execute a classe:

com.sticksfighters.game.Main
